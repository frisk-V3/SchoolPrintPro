# SchoolPrintPro

C#およびWindows Formsを用いたプリントレイアウトエンジンの学習用プロジェクトです。
GDI+による描画処理や、Win32 APIを利用したユーザーインターフェースの実装パターンの習得を目的としています。

## プロジェクトの概要
本プロジェクトは、テキスト、図形、画像をキャンバス上に自由に配置し、印刷およびPDF出力を制御するための基礎的なエディタ機能を実装しています。

## 実装されている主な機能
- 動的レイアウト：テキスト、ボックス、画像の自由な配置と管理
- ファイルインポート：外部画像ファイルのドラッグ＆ドロップ対応
- キャンバス操作：マウスホイールによるズーム、スペースキー併用によるパン（画面移動）
- ネイティブ連携：Win32 API（user32.dll）を用いたウィンドウ操作の制御
- 高度な描画：GDI+を用いたアンチエイリアス処理および座標変換（Scale/Translate変換）

## 学習の主なトピック
- System.Drawing (GDI+) を用いた動的な2Dグラフィックス描画の実装
- P/Invoke によるアンマネージドコード（Win32 API）の呼び出し
- マウス・キーボード入力を組み合わせたインタラクティブなUIコントロールの構築
- ワールド座標とデバイス座標の相互変換ロジックの理解

## 開発環境
- 言語：C#
- フレームワーク：.NET (Windows Forms)
- 依存ライブラリ：System.Drawing, System.Windows.Forms, System.Runtime.InteropServices

## ライセンス
本プロジェクトは学習用として公開されています。
